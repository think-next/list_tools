# List Tools - H3 地理网格工具集

一个功能强大的 Chrome 浏览器扩展，提供专业的 H3 地理网格索引计算和可视化工具。

## 🌟 主要功能

### **H3 网格索引计算**
- **坐标转换**：快速将经纬度坐标转换为 H3 网格索引
- **网格计算**：输入 H3 索引，获取详细的网格信息
- **支持所有分辨率**：从 0 到 15 级网格级别

### **网格扩圈功能**
- **智能扩圈**：计算任意指定距离内的所有网格单元
- **详细统计**：显示扩圈包含的网格数量
- **完整顶点坐标**：提供所有网格的顶点坐标数据

### **精确的地理数据**
- **中心点坐标**：高精度显示网格中心点（保留6位小数）
- **面积计算**：计算网格的实际面积（平方米）
- **边长统计**：显示网格的最长和最短边长
- **半径分析**：提供顶点半径和边心半径（最长/最短）
- **父单元追溯**：查看上一级网格单元

### **顶点数据导出**
- **顶点串**：单个网格的完整顶点坐标
- **批量坐标导出**：扩圈所有网格的顶点坐标
- **一键复制**：便捷的数据复制功能

### **自定义工具管理**
- **创建工具**：添加您常用的网页工具快捷方式
- **个人工具集**：打造专属的工作工具集合
- **快速访问**：一键直达您常用的在线工具

## 📦 安装方法

### 从 Chrome Web Store 安装（推荐）
1. 访问 [Chrome Web Store](https://chrome.google.com/webstore)
2. 搜索 "List Tools - H3 网格工具"
3. 点击「添加至 Chrome」完成安装

### 开发者模式安装
1. 下载或克隆项目到本地
2. 打开 Chrome，访问 `chrome://extensions/`
3. 右上角开启「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择项目的 `ext` 目录

## 🚀 快速开始

### 计算 H3 网格
1. 点击浏览器工具栏中的扩展图标
2. 在「坐标计算」标签中：
   - 输入经纬度（格式：纬度,经度，如 `37.775,-122.418`）
   - 选择网格级别（0-15）
   - 设置扩圈数量（0-10）
3. 立即查看详细的网格信息

### 查询网格信息
1. 切换到「网格计算」标签
2. 输入 H3 网格索引（如 `8a1fb46622dffff`）
3. 查看该网格的所有详细信息

### 添加自定义工具
1. 点击「新增工具」卡片
2. 输入工具名称和跳转链接
3. 保存后即可快速访问
